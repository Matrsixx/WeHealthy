#Tutor Include Navbar

1.  Copy script dulu
    ```
    <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
    <script>
        $(function(){
            $("#navbar").load("include/navbar.html"); 
        });
    </script>
    ```

2. Paste script di file .html nya
![Alt text](image.png)

3. Copy div ini 
    ```
    <div id="navbar"></div>
    ```
    
4. Paste div nya di body
